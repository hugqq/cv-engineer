## 项目简介

立志做一名优秀的Java CV工程师

## 开发环境

- **JDK 17 +**
- **Maven 3.8 +**
- **IntelliJ IDEA  2023.1 +**
- **Mysql 5.7 +**
- **Nodejs 14.9.0+ **
- **Docker desktop**
